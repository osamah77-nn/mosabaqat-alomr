مسابقة العمر | Mosabaqat Alomr

وصف المشروع

منصة مسابقات وجوائز إلكترونية تعمل بنظام السحب والدفع.

طريقة التشغيل

1. تشغيل السيرفر:
   npm install
   npm start

2. فتح المتصفح:
   http://localhost:5000

3. شغل التطبيق:
   ```
   flutter run
   ```

   التطبيق سيعرض نص "مسابقة العمر" في المنتصف.

## التطوير المستقبلي

المشروع جاهز لإضافة:
- تسجيل المستخدمين
- شراء الأسهم
- قاعدة بيانات
- لوحة تحكم المسؤول

عنوان السيرفر محفوظ في `frontend/lib/config.dart` لسهولة التغيير.
