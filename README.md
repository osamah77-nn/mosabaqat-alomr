وصف المشروع

منصة مسابقات وجوائز إلكترونية تعمل عبر الويب بنظام السحب والدفع.

طريقة التشغيل

1. تثبيت الاعتمادات:
   ```
   cd backend
   npm install
   ```

2. تشغيل السيرفر:
   ```
   npm start
   ```

3. فتح المتصفح:
   ```
   http://localhost:5000
   ```

الواجهة الرسمية للمشروع هي واجهة الويب الموجودة في `backend/public/index.html`.
