// تصدير جميع الأرقام الممنوحة مع بيانات أصحابها إلى ملف CSV function exportAllTicketsCSV() { const users = getUsers(); let rows = [['رقم السحب','اسم المستخدم','اسم العائلة','البريد الإلكتروني']]; users.forEach(u => { const tickets = getUserTickets(u.email); if (tickets.length === 0) { // حتى لو لم يكن لديه أرقام، أضف صف باسم المستخدم والبريد بدون رقم rows.push(['', u.username, u.family || '', u.email]); } else { for (let i = 0; i < tickets.length; i++) { rows.push([tickets[i], u.username, u.family || '', u.email]); } } }); let csvContent = rows.map(r => '"'+r.join('","')+'"').join('\n'); // إنشاء ملف وتحميله const blob = new Blob([csvContent], {type: 'text/csv'}); const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = 'all_tickets_database.csv'; document.body.appendChild(a); a.click(); setTimeout(() => { document.body.removeChild(a); URL.revokeObjectURL(url); }, 500); }
